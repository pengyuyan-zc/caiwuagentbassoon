# 开票主体税务凭证生成

当用户请求"生成开票税务凭证"、"生成税务凭证"、"X月开票税务凭证"、"开票主体税务凭证"、"发票税务凭证"时，立即使用此 skill。

## 凭证分录规则

### 全额开票（3条分录）
```
借 1122 应收账款 = 开票总额（填客户编码/名称）
贷 6001.xx.xx 主营业务收入 = 不含税金额
贷 2221.01.02 销项税额 = 增值税金额（≠0 时才写）
```

### 差额开票（4条分录）
```
借 1122 应收账款 = 开票总额（填客户编码/名称）
贷 6001.xx.xx 主营业务收入 = 不含税金额
贷 2221.01.02 销项税额 = 增值税金额（≠0 时才写）
贷 6001.xx.xx 主营业务收入 = 差额开票金额（>0 时才写）
```

### 无税额仅差额（2条分录）
```
借 1122 应收账款 = 开票总额（填客户编码/名称）
贷 6001.xx.xx 主营业务收入 = 差额开票金额
```

## 业务类型 → 科目映射

| 内部类型分类 | 主营业务收入科目 |
|------------|--------------|
| 派遣业务 | 6001.02.00 |
| 代理业务 | 6001.03.01 |
| 承揽业务 / 小时间承揽 | 6001.04.01 |
| 其他未识别 | 6001.04.01（默认） |

## 科目编码

| 科目名称 | 编码 |
|---------|------|
| 应收账款 | 1122 |
| 派遣收入 | 6001.02.00 |
| 代理收入 | 6001.03.01 |
| 承揽收入 | 6001.04.01 |
| 销项税额 | 2221.01.02 |

## 摘要生成

摘要 = 客户简称 + YYYYMM + "月开票"

客户简称从客户明细表匹配，未命中则去掉发票名称的地区前缀。

## 调用方式

请调用 `agent.py` 中的 `invoke_invoice_voucher` 函数：

```python
from agent import invoke_invoice_voucher

result = invoke_invoice_voucher(
    input_file="开票主体.xlsx",   # str 或 Path，开票主体Excel文件路径
    output_dir="输出目录"          # 可选，默认为输入文件同目录
)
```

## 返回格式

```python
{
    'success': bool,
    'output_file': '凭证输出_20260416_120000.xlsx',
    'voucher_count': 15,   # 凭证数量
    'line_count': 45,      # 分录数量
    'message': '生成完成！共 15 张凭证，2.3 秒'

```

## 依赖要求

- Python 3.8+
- openpyxl
- 凭证模板（金蝶云凭证明细模板.xlsx）在 skill 目录或桌面
- 客户明细（客户明细2026*.xlsx）在 skill 目录或桌面

## 错误处理

- 开票总额 <= 0 或单元格含 #VALUE! → 自动跳过该行
- 客户明细未找到 → 警告打印未匹配客户名，继续生成
- 科目表/模板未找到 → 返回 success: False

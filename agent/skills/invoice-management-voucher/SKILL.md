# 开票管理帐凭证生成

当用户请求"生成开票管理帐凭证"、"生成开票管理帐凭证"、"X月开票管理帐凭证"、"开票主体管理帐凭证"、"管理帐发票凭证"时，立即使用此 skill。

## 凭证分录规则

每张发票（审批编号）生成一张凭证，含 2-3 条分录：

### 广东谊丰（账簿 501）
```
借 1122.xx  应收账款（按内部类型选子目） = 开票总额（填客户编码/名称）
贷 6001.xx  主营业务收入（按内部类型选子目） = 不含税金额
贷 2221.01.02 销项税额（增值税金额 > 0 时） = 增值税金额
```

### 非广东（账簿 102）
```
借 1122.xx  应收账款（按内部类型选子目） = 开票总额（填客户编码/名称）
贷 6001.xx  主营业务收入（按内部类型选子目） = 不含税金额
贷 2221.01.02 销项税额（增值税金额 > 0 时） = 增值税金额
```

## 业务类型 → 科目映射

| 内部类型分类 | 应收账款子目 | 主营业务收入科目 |
|------------|------------|---------------|
| 派遣业务 | 1122.02 | 6001.01 |
| 代理业务 | 1122.03 | 6001.02 |
| 承揽业务 | 1122.04 | 6001.03 |
| 猎头业务 | 1122.05 | 6001.04 |
| 灵工/小时间 | 1122.11 | 6001.10 |
| 未识别 | 1122.04（默认） | 6001.03（默认） |

## 开票主体 → 账簿映射

| 开票主体 | 账簿编码 | 账簿名称 |
|---------|---------|---------|
| 广东谊丰 | 501 | 广东公司 |
| 其他 | 102 | 按主体名称 |

## 摘要生成

摘要 = 开票主体简称 + "开票--" + 客户简称 + 年月 + "月开票"

## 调用方式

请调用 `agent.py` 中的 `invoke_management_voucher` 函数：

```python
from agent import invoke_management_voucher

result = invoke_management_voucher(
    input_file="OA开票申请.xlsx",   # str 或 Path，OA 开票申请数据文件路径（必填）
    output_dir="输出目录",           # 可选，默认为输入文件同目录
    template_file=None,             # 可选，默认在管理帐目录下自动搜索含"管理"+"凭证"的 xlsx
)
```

## 文件依赖路径（供参考）

本 skill 目录完全自包含，以下文件均位于同一目录下：
- 凭证模板：`invoice-management-voucher\202510月凭证-管理.xlsx`（自动搜索含"管理"+"凭证"的 xlsx）
- 客户字典：`invoice-management-voucher\客户明细管理帐.xlsx`
- 执行脚本：`invoice-management-voucher\generate_voucher.py`（由 agent.py 通过 subprocess 调用）

## 依赖要求

- Python 3.8+
- openpyxl
- 凭证模板（金蝶云管理帐凭证明细模板.xlsx）在管理帐目录下（`pythonProject6（ruibo）\管理帐\`）
- 客户字典（客户明细管理帐.xlsx）在管理帐目录下

## 错误处理

- 审批状态 ≠ 已完成 → 自动跳过该行
- 开票总额 ≤ 0 或单元格含 #VALUE! → 自动跳过该行
- 客户字典未找到匹配 → 警告打印未匹配客户名，继续生成
- 模板文件未找到 → 返回 success: False
